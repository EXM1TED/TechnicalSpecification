В приложение есть файл конфигурации, он запомианет тот путь, который вы указали. Если вы не выбрали путь к файлам, 
то все отфильтрованные данные будут выходить в файлы, которые располагаются в папке: LogFiles. 
Папка LogFiles находится в папке с проктом
Встроенная база данных - SQLite, файл .db хранится в папке с проектом
Присутствует логирование валидации о основных операций
